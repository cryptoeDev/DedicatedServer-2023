Полностью рабочий сервер боев Warface, который поддерживает:

- PvE режимы. Рабочая AI-система, передвижение ботов, Movement, стрельба. 
- PvP режимы и конфигурации к ним. 
- Полностью восстановленная CGame структура, необходимая для сервера боя.
- Восстановленные некоторые пакеты, вернее вызовы, которые восстанавливают их: (update_achievement, setserver, mission_load, mission_update, mission_unload, telemetry_stream, getprofile, set_rewards_info)

Бонусы:
 - Конфигурации (dedicated.pak -> content folder)
 - GameScriptsC_X64.pak (необходимый для .lua скриптинга, который выполняет AI-система)
